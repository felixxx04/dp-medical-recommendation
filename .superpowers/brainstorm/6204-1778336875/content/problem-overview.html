Problem #1: No Recommendation (Empty Results)

Affected Disease Types:
• Viral diseases — 病毒感染, 上呼吸道感染 (no specific antiviral in DB)
• Rare/orphan diseases — 肺纤维化 (limited treatment options)
• Non-vocab diseases — mapped to wrong proxy → all candidates excluded
Root Cause Chain:
Disease → mapper uses proxy word → proxy word matches only unrelated drugs → SafetyFilter excludes them → 0 results

Problem #2: Wrong/Inappropriate Recommendation

Failure Modes:
A. Category Mismatch: Antihypertensives for 白内障 (fixed via SafetyFilter rule #19)
B. Wrong Drug Class: Antibiotics for viral URI (fixed via rule #10)
C. DP Noise Reversal: Low-score correct drug pushed below high-score wrong drug
D. Proxy Disease Contamination: "enteritis" → "diarrhea" proxy → IBS drugs instead of anti-infectives
Remaining After Fixes: 27 diseases APPROPRIATE→PARTIALLY degraded
e.g. 高血压 (antihypertensives are correct but ranking suboptimal),
癫痫 (anti-epileptics correct but DP noise affects ordering), 肺炎 (antibiotics correct but broad-spectrum vs narrow)
Key Question: How should we redesign the pipeline to address both gaps — diseases with no path to recommendation, and recommendation paths that lead to wrong drugs?
